One of my most recent projects would be a Python DNS Network sniffer. The goal of this project was to gain experience an learn a little bit more about cybersecurity. This program utilizes many libraries to include the main library called Scapy.

Scapy is a powerful Python library used for network packet manipulation and analysis. It allows users to create, send, and capture network packets, making it an essential tool for network security and analysis tasks.
